# Installation Guide

## Prerequisites

- Python 3.8 or higher
- pip package manager
- Git (optional, for development)

## Installation Methods

### 1. Using pip

```bash
pip install sacp
```

### 2. From Source

```bash
git clone https://github.com/yet-studio/sacp.git
cd sacp
pip install -e .
```

## Verify Installation

```bash
sacp --version
```

## IDE Extensions

### VSCode Extension

1. Open VSCode
2. Go to Extensions (Ctrl+Shift+X)
3. Search for "SafeAI CodeGuard"
